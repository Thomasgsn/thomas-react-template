### How to use

Get my template
```
  git clone https://github.com/Thomasgsn/thomas-react-template.git
```

Install dependencies
```
# with pnpm
  cd thomas-react-template
  pnpm i
```
```
# with yarn
  cd thomas-react-template
  yarn
```

To run
```
# with pnpm
  pnpm dev
```
```
# with yarn
  yarn dev
```
